## System requirements
* Composer (global installed);
* PHP >= 8.1;
* GNU Make >= 4.3;
* npm >= 18.0;
* Node >= 16.0;
* PostgreSQL >= 14.8;
* Docker >= 24.0.

<br>

## Install
```sh
git clone https://github.com/Kolalexx/php-project-57.git
cd php-project-57
make setup
```

<br>

### Start
```sh
make start
```

> For local start open localhost:8000 in your browser
